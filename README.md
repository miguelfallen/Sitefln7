## 🚀 Overview

**Sitefln7** is a high-performance interactive web assistant powered by generative AI. Built with a focus on the gaming community, it utilizes Google's **Gemini API** to provide precise and context-aware responses.

> [!IMPORTANT]
> **Gaming Guardrails:** This AI is specifically fine-tuned for gaming topics. If a user asks questions outside the gaming domain, the system is programmed to politely decline and maintain focus on its core purpose.

## 🛠️ Tech Stack

- **Backend:** Python & Flask
- **AI Engine:** Google Gemini API (Generative AI)
- **Frontend:** HTML5, CSS3, and Vanilla JavaScript
- **Operations:** CI/CD practices and dependency management via `requirements.txt`
- **Hosting:** Render

## 🧠 Key Features

* 🎮 **Context Awareness:** Specialized knowledge in game mechanics, lore, and strategy.
* 🚫 **Topic Filtering:** Automatic detection of non-gaming queries.
* ⚡ **Fast Response:** Optimized communication between Flask and the Gemini API.
* 🌐 **Cloud Ready:** Fully deployed and operational in a production environment.
